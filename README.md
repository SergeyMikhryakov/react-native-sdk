# mindbox-sdk

SDK for integration React Native mobile apps with Mindbox

## Installation

```sh
npm install mindbox-sdk
```

## Usage

```js
import MindboxSdk from "mindbox-sdk";

// ...

const result = await MindboxSdk.multiply(3, 7);
```
